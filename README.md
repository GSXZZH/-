# 王朴中学社区网站

## 项目简介

王朴中学社区网站是一个为学生、教师和家长提供交流和资源分享的平台。网站采用红色主题，包含用户系统、聊天功能、资源分享和个人设置等功能模块。

## 功能模块

### 1. 用户系统
- 登录/注册功能
- 使用localStorage永久保存用户数据
- 默认用户：
  - 管理员：admin/admin123
  - 学生：student1/student123
  - 教师：teacher1/teacher123

### 2. 聊天功能
- 支持用户之间实时聊天
- 消息历史记录保存
- 按日期分组显示消息

### 3. 资源分享
- 支持文件上传和下载
- 资源列表展示
- 文件类型识别和图标显示

### 4. 个人设置
- 支持更改头像
- 个人信息编辑
- 密码修改

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- localStorage (本地数据存储)

## 项目结构

```
wangpu_school_community/
├── index.html          # 登录/注册页面
├── home.html           # 首页
├── chat.html           # 聊天页面
├── resources.html      # 资源页面
├── settings.html       # 个人设置页面
├── ziyuan/             # 资源存储文件夹
└── README.md           # 项目说明
```

## 使用指南

1. 打开 `index.html` 页面进行登录
2. 登录后自动跳转到首页
3. 通过导航栏切换不同功能模块
4. 所有数据存储在浏览器本地，刷新页面不会丢失数据

## 注意事项

- 头像和资源文件上传后仅在本地存储，刷新页面后可继续访问
- 聊天功能仅支持同一浏览器内的用户之间通信
- 建议使用现代浏览器（Chrome、Firefox、Edge等）访问网站
